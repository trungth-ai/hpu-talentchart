# TalentChart

Nền tảng SaaS tuyển dụng & đánh giá nhân sự tích hợp Eastern Personality Assessment (EPA) — dự án của HPU AI Lab.

Scaffold tạo ngày 2026-07-02 từ framework `hpu-dev` v2.0 (`--type=saas`, port `8003`).

**Đọc theo thứ tự:**
1. `docs/HUONG-DAN-TRIEN-KHAI-CLAUDE-CODE.md` — hướng dẫn triển khai từng bước bằng Claude Code
2. `CLAUDE.md` — quy ước bắt buộc của project (Claude Code tự đọc file này)
3. `docs/PLAN.md` — sprint đang chạy
4. `docs/DECISIONS.md` — các quyết định kiến trúc đã chốt (ADR)

Lưu ý: thư mục này được scaffold trong môi trường sandbox nên chưa có `git init` thật (giới hạn quyền trên ổ đĩa mounted) — xem bước 2 của file hướng dẫn để khởi tạo git trên máy dev thật.
